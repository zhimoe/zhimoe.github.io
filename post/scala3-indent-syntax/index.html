Scala3 缩进语法总结表

2022-02-02 编程 #scala3 #cheatsheet

Scala 3 在语法上面新增了一种 Python 的缩进格式，两种格式都可以使用。但是目前部分情况还是需要使用括号。
个人对新语法是支持的。缩进可以极大地提供代码的可读性和整洁，最大的体会就是 SparkStreaming 的 rdd 处理代码，新手容易写出十几个}括号嵌套代码。
当然缺点是缩进不利于代码复制和格式化。
