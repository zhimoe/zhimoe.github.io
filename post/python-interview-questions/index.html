    v[1] = 4

本意其实是想得到一个函数列表[0x,1x,2*x]， 但是 Python’s closures are late binding. This means that the values of variables used in closures are looked up at the time the inner function is called. 解决方案是偏函数partial

from functools import partial
def fix_fn_for():
    f = [partial(lambda y, x: y * x, x=i) for i in range(3)]
    print(f[0](1), f[1](1), f[2](1))

或：

fl=[lambda x, i=i: x*i for i in range(3)]
