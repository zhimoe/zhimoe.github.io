隐式参数

//隐式参数是在调用时可以自动填充的参数, 需要在调用范围内（scope)有一个隐式变量可供填充.
def addInt(i:Int)(implicit n: Int) = i + n


//需要提供一个隐式变量n
implicit val sn = 1
addInt(2) // 3

//如果有两个满足类型的隐式变量,则在编译addInt(2)时报错
